# Weighbridge Client Agents

Two independent agents for client-site deployment. Install one or both based on client needs.

## Files to Copy to Client PC

```
agents/
  scale_agent.py          # Weighbridge scale agent
  camera_agent.py         # IP camera agent
  requirements.txt        # Python dependencies (pyserial + requests)
```

## Quick Setup

```powershell
# 1. Create folder
mkdir C:\weighbridge-agent
copy scale_agent.py C:\weighbridge-agent\
copy camera_agent.py C:\weighbridge-agent\
copy requirements.txt C:\weighbridge-agent\

# 2. Install dependencies
cd C:\weighbridge-agent
pip install -r requirements.txt

# 3. Setup each agent (interactive wizard)
python scale_agent.py --setup
python camera_agent.py --setup

# 4. Test camera (saves test images locally)
python camera_agent.py --test

# 5. Run interactively (for testing)
python scale_agent.py
python camera_agent.py

# 6. Install as Windows services (auto-start on boot)
python scale_agent.py --install
python camera_agent.py --install
```

## Windows Services

| Service Name | Port | What It Does |
|---|---|---|
| WeighbridgeScaleAgent | 9002 | Reads COM port, pushes weight to cloud |
| WeighbridgeCameraAgent | 9003 | Polls for events, captures + uploads snapshots |

## Health Check

```
http://localhost:9002    # Scale agent status
http://localhost:9003    # Camera agent status
```

## Uninstall

```powershell
python scale_agent.py --uninstall
python camera_agent.py --uninstall
```

## Config Files (generated by --setup)

- `scale_config.json` — cloud URL, tenant, agent key, COM port, baud rate
- `camera_config.json` — cloud URL, tenant, agent key, camera URLs
