What is WeighBridge Pro?

An end-to-end digital operations platform purpose-built for stone crushers, quarry operators, and aggregate suppliers in India.

⚖️

Smart Weighbridge Operations

Two-stage digital weighment with real-time scale integration, automatic token generation, and IP camera snapshot capture — eliminating manual recording errors.

🧾

GST-Compliant Invoicing

Automatic invoice creation from tokens, B2B/B2C GST invoices, GSTR-1 and GSTR-3B reports, gap-free invoice numbering, and full Tally Prime integration.

📊

Business Intelligence

Real-time dashboard, P&L reports, stock summaries, outstanding ageing analysis, compliance tracking, and multi-channel notification system — all in one platform.

19+ Business Use Cases

WeighBridge Pro covers every workflow from the weighbridge to the boardroom — no third-party tools needed.

⚖️Weighbridge Operations

  • Sale token: Gross → Tare → auto net weight
  • Purchase token: Loaded → Unloaded → net weight
  • Real-time serial scale integration
  • Manual weight entry fallback
  • Token search by vehicle, party, date
  • Cancel / supplement token workflow

📷IP Camera Capture

  • Front View + Top View cameras at weighment
  • Auto-capture on second weight — non-blocking
  • 3-retry logic with 5-second timeout
  • Per-camera status in real-time dialog
  • Lightbox image viewer on completed tokens
  • Admin: configure URL, credentials, live test

🧾Sales Invoicing

  • B2B GST invoice with CGST/SGST or IGST
  • B2C walk-in (no GSTIN required)
  • Auto-created from token on completion
  • Gap-free invoice numbering per FY
  • Discount (flat or percentage), TCS, freight
  • PDF generation, cancel, finalise workflow

🛒Purchase Invoicing

  • Auto-created from purchase token
  • Manual purchase invoice (no token)
  • Separate purchase numbering sequence
  • Party-specific product rates
  • ITC tracking for GSTR-3B
  • Tally sync as Purchase voucher

📋Quotations

  • Create, send, accept, reject quotations
  • Valid-to date with expiry tracking
  • One-click conversion to sales invoice
  • PDF download and share
  • Date range filter and status tracking
  • Configurable quotation prefix

💵Payment Management

  • Sales payment receipts (Cash, UPI, NEFT, Cheque)
  • Purchase payment vouchers (outgoing)
  • Partial payment support per invoice
  • Running party balance updated in real-time
  • Payment status: Unpaid → Partial → Paid
  • Link multiple payments to one invoice

📈Outstanding & Ledger

  • Party-wise running debit/credit ledger
  • Outstanding ageing: Current, 1–30, 31–60, 61–90, 90+ days
  • Credit limit breach alerts
  • Payment terms tracking
  • Drill-down from outstanding to invoice
  • CSV export for finance team

🔗Tally Prime Integration

  • Sales & Purchase XML voucher push
  • Multi-select bulk sync with progress
  • Configurable 9 ledger names in settings
  • Vehicle, token & weight in narration
  • Bill-wise aging (BILLALLOCATIONS) for Tally reports
  • Per-party ledger name override

📑GST Reports

  • GSTR-1: B2B invoices with GSTIN details
  • GSTR-1: B2C summary & HSN-wise summary
  • GSTR-1 JSON export (GSTN portal format)
  • GSTR-3B: Section 3.1 outward supplies
  • GSTR-3B: Section 4 — eligible ITC
  • GSTR-3B: Net tax payable computation

💹Business Reports

  • Sales & Purchase register (date/party filter)
  • Weight register — token-wise tonnage
  • Profit & Loss — monthly revenue vs COGS
  • Stock summary — product-wise in/out/closing
  • Gross margin % per product
  • CSV export for all reports

👥Party & Product Master

  • Customer / Supplier / Both party types
  • GSTIN, PAN, credit limit, payment terms
  • Party-specific product price overrides
  • Product categories, HSN codes, GST rates
  • Vehicle master with tare weight history
  • Driver & transporter management

📊Live Dashboard

  • Today's token count & revenue widget
  • Total tonnage processed today
  • Outstanding balance across all parties
  • Top 5 customers by revenue
  • Recent tokens feed with status
  • Compliance alerts banner

🔐Private Invoice System (USB)

  • USB hardware key authentication
  • AES-256-GCM encrypted private invoices
  • Separate SE/NNNNN gap-free sequence
  • Non-GST invoices excluded from GST reports
  • Hourly auto-backup to USB drive
  • Private admin audit console

📁Compliance Tracker

  • Insurance, certification, license, permit records
  • Expiry date alerts: OK → Warning → Critical → Expired
  • Configurable warning & critical day thresholds
  • File upload (PDF/image) with browser preview
  • Clickable alert cards filter the table
  • Dashboard compliance banner

🔔Notifications

  • SMTP email notifications
  • SMS via MSG91 gateway
  • WhatsApp via WATI integration
  • Jinja2 templates per event type
  • Test send from settings panel
  • Delivery log with error tracking

👤User & Access Control

  • 7 roles: Admin, Operator, Sales Executive,
    Purchase Executive, Accountant, Viewer, Private Admin
  • Configurable per-role page permissions
  • Create / edit / deactivate users
  • Admin password reset for any user
  • JWT token security (8-hour expiry)
  • Full audit trail per user action

🗂️Audit Trail

  • All create/update/delete actions logged
  • Filter by action type, entity, user, date
  • Stats cards: actions by type & entity
  • Immutable record (cannot be deleted)

💾Backup & Data Import

  • One-click pg_dump database backup
  • Download backup files to local machine
  • Restore from any previous backup
  • Bulk import parties/products/vehicles from Excel/CSV
  • 10-row preview before confirm
  • Blank template download

🏢Company & FY Setup

  • Company profile: name, GSTIN, PAN, bank details
  • Customizable invoice/quotation/purchase prefixes
  • Multiple financial years (activate/switch)
  • Custom wallpaper branding
  • State-wise GST (CGST+SGST vs IGST auto-detection)
